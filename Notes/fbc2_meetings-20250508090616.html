    • [X] What is the selection for output jets in mixing and synthetic datasets ? Sent the following to Marina A: For mixed data all the input jets are mixed so the only preselection is the cut to make the nanoAOD (This is b/c the performance of mixing algorithm doesnt really scale with the number of jets) For the synthetic data I do use. a preselection to speed up the DeClustering algorithm The preselection is (event.Jet.pt >= 20) & (event.Jet.jetId >= 2) & event.Jet.leptoncleaned & (np.abs(event.Jet.eta) <= 4.7) That preselection is on the input jets used to cluster (and compute the templates) when I sample the template to make the synthetic (fake) jets I make up new pts and eta. IF the fake jets fail the pt or eta acceptance I try again with a new random number. But I only allow 8 attempts, after which point I just accept the jets so here that can be some (hopefully small) number of jets that have pt<20 (For the b-jets I require pt > 30 and eta < 2.5)
